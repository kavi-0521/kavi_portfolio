# Portfolio Theme Concept

## Reference-Derived Direction

The target portfolio style is:

- bold and headline-driven
- section-first storytelling (`About`, `Experience`, `Skills`, `Projects`, `Contact`)
- premium dark atmosphere with warm highlight accents
- card-based professional timeline and project showcase
- strong personal branding language and CTA focus

## Frontend Execution in This Project

- Cinematic gradient background with layered glow
- Sticky capsule navigation and editorial typography
- Hero section with clear personal proposition and CTA pair
- Skills marquee to mimic high-energy technology display
- Modular section architecture to keep content scalable

## Backend Content Strategy

- Portfolio content is served via an API endpoint (`/api/portfolio`)
- Contact form is wired to API (`/api/contact`)
- The API structure is MongoDB-ready and can be moved from static seed to Mongoose models
